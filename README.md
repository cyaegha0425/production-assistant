# 生产助手

阀门产品组装模板管理系统，用于管理产品物料信息和生成组装填写内容。

## 目录结构

```
生产助手/
├── README.md           # 本文件
├── 归档清单.md         # 所有产品的索引和核心数据
└── [产品名]/
    ├── 物料编码清单/   # Excel物料清单
    ├── 模板图片/       # 产品实拍图
    └── 组装模板交付.md # 填写好的模板（可直接复制使用）
```

## 核心概念

- **整机信息**：模板名称、设备类型、设备型号
- **组装事项**：每个零件的物料编码+慧飞+规格描述
- **填写格式**：直接复制 `组装模板交付.md` 中的内容到网页表单
