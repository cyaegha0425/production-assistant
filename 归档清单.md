# 生产助手知识库

阀门产品组装模板管理系统，管理产品物料信息和生成组装填写内容。

## 目录结构

```
./生产助手/
├── README.md           # 总体说明
├── 归档清单.md         # 本文件：所有产品索引
└── [产品名]/
    ├── 物料编码清单/   # Excel物料清单
    ├── 模板图片/       # 产品实拍图
    └── 组装模板交付.md # 填写好的模板（可直接复制使用）
```

## 填写规范

- **模板名称**：整机规格描述（各部件用 `/` 分隔）
- **设备类型**：下拉选择（默认：智慧球阀）
- **设备型号**：整机物料编码
- **组装事项格式**：`物料编码 慧飞 规格描述`
